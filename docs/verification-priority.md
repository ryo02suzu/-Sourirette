## 使い方
1. 上から順に、その区分の実レセプト（実家クリニック）を既存レセコン出力と突き合わせる。
2. 一致＝「検証状態」を ☑ 検証済 に。不一致＝ルール（病名/区分/コード）を修正し再テスト。
3. A→B→C を一巡したら、施設基準(D)の加算コード特定→有効化、最後に適応(E)。
4. JSON更新後は `npm run verify:plan` で本リストを再生成。
